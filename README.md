# Student Performance Analysis

This project analyzes the Student Performance dataset using Python.

## Objectives
- Load and clean the dataset
- Perform exploratory data analysis
- Answer key academic performance questions
- Visualize insights using graphs

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Average final grade (G3)
- Students scoring above 15
- Study time vs performance
- Gender-wise performance comparison

## Conclusion
The analysis provides insights into how study time and gender relate to student performance.
